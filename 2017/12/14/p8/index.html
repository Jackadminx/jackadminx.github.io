联想P8平板搞机小记

用了几个月说说感受吧

优点：性价比高，省电，运行流畅

缺点：系统简陋且有一些bug，屏幕素质一般

被windows平板折腾得够呛的我决定转向安卓平板，打算用于娱乐和阅读文档希望能简单方便一点，结果却又进了另外一个大坑。在贴吧和京东逛了许久，于是双11之前在马云家入手P8平板。其实还有一款就是京东平板，性价比更好，但是看到系统（Flyme）没法root还是放弃了。

最想吐槽的就是系统了，简陋到不行；什么桌面设置，分屏，就连自定义下拉栏都没。。。算了这些root后应该可以解决。

到了Android6.0以后有一个不错的功能：将内存卡作为内部储存来用，可以将部分应用迁移到内存卡中。但这样内存卡就不能给另外的设备使用。有一次就是忘了备份，刷机后手机电脑都打不开内存卡，搞得我只能格式化😭

备份QCN

高通备份QCN是刷机之前必做的事情，主要是设备串号序列号等信息

1.到联想社区下载对应的设备驱动和QPST，安装。

2.关闭平板，摁住右侧所以按键，进入工厂模式（此时会打开高通Diagnostics端口）数据线连接电脑

3.打开QPST Configuration 点击 add new port 选择Diagnostics端口

4.选择start clients>software download

5.backup>文件输出选择.qcn后缀>点击start

（win10系统可能无法点击start，使用tab键解决）

恢复QCN

点击restore>选择QCN文件>勾选mismatch选项>start

刷机

这款平板对应的Rom非常之少，暂时只找到国际版和国内版的固件。国际版的固件虽然自带google套件但在国内搜不到GPS。国内版在联想社区的帖子被删除了，但是在贴吧还可以找到网盘分享。

1.平板关机，按住音量上键，数据线连接电脑进入9008端口（推荐使用USB2.0接口）

2.打开QFIL，设定端口和文件

然后点击Download开始刷机

刷入TWRP和Root

刷TWRP的方法与刷固件的方法一致，这里就不重复了。

刷入成功后自动进入TWRP,点击高级>root系统就可以了

注意root后打开设置里面应用的自启动，不然可能会导致root授权无法弹出

安装Google Play服务

平板安装Open gapps（下载缓慢记得科学上网），下载对应版本卡刷包，进入TRWP安装

文件位于/sdcard/download/opengapps目录下

如遇到服务已经停止运行，进入系统设置授予google play服务权限。
